///////////////////////////////
 Run Configurations
////////////////////////////////

To run the project,

1. Clone to the github repo
2. git pull
3. Open as existing java project in your IDE
4. All the java files are in src folder, Right click on whichever file you want to run.
         OR
    Navigate to the SRC in Terminal and run with "java" command
